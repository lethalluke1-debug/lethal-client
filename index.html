// what's actually real

Everything here does what it says

No mockups pretending to be features. Every button below talks to a real service.

🔑

Real Microsoft sign-in

Opens your actual browser for a genuine Microsoft/Xbox login. Ownership of Minecraft: Java Edition is checked for real — no login without a real purchase.

📦

Real Modrinth mods

Every mod in the library is pulled live from Modrinth's public API and installed as an actual file in your instance — not a fake progress bar.

⚙️

Real Fabric install

Downloads the real vanilla client, libraries, assets, and Fabric loader straight from Mojang and Fabric's own servers, then actually launches the game.

🗂️

Per-version instances

Every Minecraft version gets its own separate mods folder, so switching versions never mixes incompatible mods together.

🌐

Live server status

Add any server address and see real, live player counts and icons — the same way the vanilla multiplayer screen works.

🔄

Updates itself

New versions roll out automatically in the background. Open the app, and it's already up to date.

// setup

Up and running in a few minutes

1

Download the installer

Grab the latest version below and run it — installs like any other Windows program.

2

Sign in with Microsoft

Click the account icon, sign in with the account that owns Minecraft: Java Edition.

3

Pick your mods

Browse the Mods page, install whatever you want for whichever Minecraft version you're playing.

4

Play

Hit Play. That's it — vanilla or modded, your call.
